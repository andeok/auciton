## Just Auction

모든 사용자가 공정한 규칙으로 경매에 참여하는 실시간 경매 서비스. 멤버십 혜택은 비용/편의 영역으로 제공하며 다음 3가지를 1차 목표로 구현합니다:
- 수수료 할인: 등급별 수수료율(rate)·상한(cap) 적용
- 결제 유예 연장: 낙찰 후 결제 마감(paymentDueAt) 시간 등급별 차등
- 자동입찰 슬롯: 프록시 입찰(Proxy Bidding) 룰 등록 개수 등급별 제한
